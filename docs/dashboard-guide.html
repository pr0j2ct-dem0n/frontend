전체 화면 구성 개요

대시보드는 5개 영역으로 구성됩니다 — 헤더 / KPI 카드 / 지도+상황패널 / 시계열 차트 / 데이터 출처

A. 헤더 (상단 고정 바)

서비스 이름, 실시간 상태, 데이터 기준 시각, 전체 위험 등급을 표시합니다. 화면 스크롤 시에도 항상 상단에 고정됩니다.

🔴

B. KPI 카드 4개

현재 강우량, 평균 하수관 수위, 위험 구간 수, 최고 위험 지역을 한눈에 파악합니다. 카드 클릭 시 상세 페이지로 이동합니다.

🗺

C. 서울시 위험 구간 현황 (지도)

Leaflet 기반 지도에 위험 구간, 빗물펌프장, 공공하수처리시설을 레이어별로 표시합니다. 마커 클릭/오버 시 상세 정보가 나타납니다.

🔔

D. 실시간 상황 패널

현재 위험도 순으로 구간 목록을 나열합니다. DANGER 발생 시 상단에 긴급 경고 배너가 표시되며 1분마다 자동 갱신됩니다.

📈

E. 시계열 분석 차트 (3개)

강우량 추이, 하수관 수위 변화, 위험도 상위 3개소의 시간별 점수 변화를 차트로 보여줍니다.

📋

F. 데이터 출처 및 위험도 산정 기준

사용 데이터 출처(서울 열린데이터광장, 공공데이터포털)와 위험 점수 산정에 사용되는 9가지 요소를 안내합니다.

데이터는 60초마다 자동 갱신됩니다. 헤더 우측의 새로고침 버튼으로 즉시 갱신할 수도 있습니다.
2

헤더 — 상단 고정 바

화면 최상단에 항상 고정되며, 시스템 식별 정보·실시간 상태·전체 위험 등급을 표시합니다.

1
2
3
4
5
6
1
서비스 로고 · 이름
파란색 아이콘과 함께 "서울시 하수도 막힘 위험 모니터링" 서비스명이 표시됩니다.
하단에는 영문 시스템 명칭이 보조 텍스트로 표시됩니다.
2
실시간 상태 표시
초록색 점이 깜빡이며 "실시간 하수도 위험 모니터링 상황판" 문구가 표시됩니다.
데이터가 실시간으로 연결·갱신 중임을 시각적으로 전달합니다.
3
공공데이터 출처 배지
"서울시 공공데이터 기반" 배지로 데이터 신뢰성을 표시합니다.
PC 화면(lg 이상)에서만 표시되어 헤더 공간을 효율적으로 사용합니다.
4
데이터 기준 시각
현재 표시 중인 데이터의 기준 시각을 시:분:초 단위로 보여줍니다.
사용자가 데이터의 최신성을 즉시 확인할 수 있습니다.
5
새로고침 버튼 · 마지막 갱신 시각
클릭 시 전체 데이터를 즉시 새로고침합니다.
버튼 옆에 마지막으로 데이터를 불러온 시각이 함께 표시됩니다.
6
전체 상태 배지
서울시 전체의 현재 위험 등급을 4단계로 표시합니다:
위험  경고  주의  안전
DANGER 단계면 배지가 깜빡입니다.
3

KPI 카드 4개 — 핵심 지표 한눈에 보기

가장 중요한 4가지 수치를 카드 형태로 표시합니다. 각 카드 클릭 시 해당 항목의 상세 페이지로 이동합니다.

1
2
3
4
1
현재 강우량 (10분)
최근 10분간 측정된 강우량 (mm)

• 파란색 구름 아이콘
• 전 시간 대비 증감률 표시
• 클릭 → 강우량 상세 페이지
• 증가 시 빨간 ▲ 표시
2
평균 하수관 수위
서울시 전체 센서의 평균 수위 (m)

• 청록색 아이콘
• 수위 변화율(%) 표시
• 클릭 → 하수관 수위 상세 페이지
• 수위 상승 시 위험 신호
3
위험 구간 수
현재 감시 중인 위험 구간 수 (개소)

• 5개 이상: 주황색으로 변경
• 7개 이상: 강조 하이라이트
• 클릭 → 위험 구간 전체 목록
• "현재 모니터링 중" 부지표 표시
4
최고 위험 지역
위험 점수가 가장 높은 행정동

• 빨간 위치 핀 아이콘
• 위험 등급 배지 표시
• DANGER 시 배지가 깜빡임
• 클릭 → 해당 지역 상세 분석
카드 좌측의 색상 줄(border-left)이 위험 단계를 나타냅니다: 파란색 → 강우량, 청록색 → 하수관 수위, 주황/빨간색 → 위험 구간/지역
4

서울시 위험 구간 현황 — 지도

Leaflet 기반 인터랙티브 지도로, 위험 구간·빗물펌프장·공공하수처리시설을 레이어별로 표시합니다.

1
2
3
4
5
1
레이어 토글 버튼 (지도 우상단)
3개 레이어를 켜고 끌 수 있는 버튼입니다.
• 위험구간 — 기본 ON (빨간 원형 마커)
• 빗물펌프장 — 기본 ON (파란 점 마커)
• 공공하수처리시설 — 기본 OFF (청록 사각 마커)
활성 레이어는 버튼이 강조됩니다.
2
위험 구간 마커 (원형)
위험 등급에 따라 색상이 다릅니다:
위험  경고  주의  안전
원의 크기가 위험 점수(0~100점)에 비례합니다.
마우스 오버 시 Tooltip: 지역명·등급·강우량·수위·용량% 표시
3
DANGER 전용 경고 아이콘 ⚠
위험(DANGER) 단계 구간에는 원형 마커 위에 빨간 ⚠ 아이콘이 추가로 표시됩니다.
즉각 대응이 필요한 위치를 직관적으로 식별할 수 있습니다.
4
빗물펌프장 마커 (파란 점)
배수 펌프장 위치를 파란 원형 마커로 표시합니다.
클릭 시 Popup: 펌프장명, 소재구, 연계 하천명, 시간당 처리 용량(m³/hr) 표시
5
범례 (지도 좌하단)
위험도 4단계 색상 + 펌프장·하수처리시설 마커 의미를 안내합니다.
공공하수처리시설 레이어 활성 시 청록 사각형 항목이 추가됩니다.
공공하수처리시설 마커 (레이어 ON 시)
●
운영중 시설 (청록 사각형)
클릭 시 Popup: 시설명, 주소, 일일 처리용량(㎥/일), 처리방식, 운영상태 표시
●
점검중 시설 (회색 사각형)
운영 중단 상태를 회색으로 표시. 마우스 오버 시 시설명·처리용량 Tooltip
5

실시간 상황 패널

위험 점수 순으로 정렬된 구간 목록을 실시간으로 표시하며, DANGER 발생 시 긴급 배너가 나타납니다.

1
긴급 경고 배너 (DANGER 발생 시에만 표시)
빨간 배너로 "긴급 경고 — 위험 구간 N개소 발생"을 표시합니다.
흰색 점이 깜빡이며 긴박감을 전달합니다.
DANGER 단계 구간이 0개면 배너 자체가 사라집니다.
2
위험 단계별 카운트 태그
패널 상단에 현재 위험 단계별 구간 수를 색상 태그로 표시합니다:
위험 N  경고 N  주의 N
전 구간 안전 시에는 전 구간 안전으로 표시
3
구간 카드 (위험 점수 순 정렬)
각 카드에 표시되는 정보:
• 순위 번호 — 위험도 순위
• 지역명 + 구 — 어느 동/구인지
• 위험 등급 텍스트 + 점수 (0~100점)
• 위험 원인 목록 — 구체적 원인 (예: "하수관 수위 85% 초과")
• 하단 지표 — 강우량(mm) / 수위(m) / 용량(%)
카드 배경색: 위험=빨강, 경고=주황, 주의=노랑, 안전=흰색
4
하단 자동 갱신 안내
패널 최하단에 "위험 점수 기준 내림차순 정렬 · 1분 간격 자동 갱신" 문구가 고정 표시됩니다.
목록은 스크롤 가능하며 최신 데이터가 자동 반영됩니다.
6

시계열 분석 차트 — 3개

강우량·하수관 수위·위험도를 시간 흐름에 따라 시각화하여 추세를 파악합니다.

1
2
3
1
시간대별 강우량
막대 (파란색): 10분 단위 강우량(mm)
선 (보라색): 1시간 누적 강우량(mm)

두 지표를 겹쳐 보여 집중호우 여부를 즉시 파악합니다. 마우스 오버 시 시각·수치 Tooltip 표시
2
시간대별 하수관 수위
선 + 면적 채움: 시간별 평균 수위(m)
• 최신 수위 ≥ 2.0m → 빨간색으로 전환
• 최신 수위 < 2.0m → 주황색 유지
• 위험수위 기준: 2.0m 표시
수위가 오르면 막힘·역류 위험 신호
3
위험도 추이 (상위 3개소)
현재 위험 점수 상위 3개 구간의 시간별 점수(0~100점) 추이를 3개 선으로 표시합니다.
• X축: 06:00 ~ 현재
• Y축: 위험 점수
각 선에 마우스 오버 시 지역명·점수 Tooltip
i
공통 기능
• 반응형 크기 자동 조절
• 차트 상단 범례 클릭 시 데이터셋 토글
• 어두운 테마 Tooltip (배경 #1e293b)
• 가로 3열 배치, MD 이상 화면에서 나란히 표시
7

데이터 출처 및 위험도 산정 기준

대시보드 최하단에 위치하며, 데이터 출처와 위험 점수 산정 방법을 투명하게 안내합니다.

1
2
3
4
1
서울 열린데이터광장 카드 (파란색)
사용 중인 서울시 공공 API 5종을 목록으로 표시합니다:
• 하천 수위 현황 — 위험 보조 판단
• 하수관로 수위 현황 — 막힘·역류 위험 판단
• 강우량 정보 — 10분 단위 분석
• 강수량 현황 — 지역별 패턴 분석
• 빗물펌프장 공간정보 — 배수 인프라 위치
2
공공데이터포털 카드 (청록색)
한국환경공단 제공 공공하수처리시설 현황 데이터 사용 안내.
• 태그: 하수처리시설 / 처리용량 / 환경인프라 / 대응시설 참고
• 하단 안내: "지도에서 공공하수처리시설 레이어를 켜서 위치를 확인할 수 있습니다"
3
위험도 산정 요소 카드 (회색)
위험 점수 계산에 사용되는 9가지 요소 목록:
①~⑥ 서울시 데이터 기반 요소
⑦~⑨ 공공하수처리시설 기반 요소 (청록 표시)
번호 색상으로 데이터 출처를 구분합니다.
4
하단 주석 바
• 위험도 산정 알고리즘 버전 (ver 1.1)
• 데이터 갱신 주기: 1분
• "위험도 계산은 백엔드에서 수행하며, 프론트엔드는 결과를 시각화합니다" 면책 문구
데이터 신뢰 범위를 사용자에게 명확히 전달합니다.
위험도 점수(0~100)는 백엔드 알고리즘이 계산하고, 이 대시보드는 결과를 시각화만 합니다. 프론트엔드에는 별도의 위험도 계산 로직이 없습니다.
