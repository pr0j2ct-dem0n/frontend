전체 화면 구성 개요

2

헤더 — 상단 고정 바

화면 최상단에 항상 고정되며, 시스템 식별 정보·실시간 상태·전체 위험 등급을 표시합니다.

3

KPI 카드 4개 — 핵심 지표 한눈에 보기

현재 강우량 / 평균 하수관 수위 / 위험 구간 수 / 최고 위험 지역 — 카드 클릭 시 상세 페이지로 이동

4

서울시 위험 구간 현황 — 지도

Leaflet 기반 인터랙티브 지도 — 위험도별 원형 마커, DANGER 경고 아이콘, 범례 포함

5

실시간 상황 패널

위험 점수 순으로 정렬된 구간 목록 — 상위 3개 기본 표시, 더 보기 버튼으로 전체 확인 가능

6

시계열 분석 차트 — 3개

시간대별 강우량 / 하수관 수위 추이 / 위험도 상위 3개소 점수 변화

7

데이터 출처 및 위험도 산정 기준

서울 열린데이터광장 API 5종 출처 및 위험 점수 산정 요소 6가지 안내

8

클릭 시 이동하는 상세 페이지 목록

KPI 카드 클릭 시 이동 — /rainfall · /sewer-level · /risk-zones · /highest-risk-area

/rainfall — 강우량 상세
/sewer-level — 하수관 수위 상세
/risk-zones — 위험 구간 전체 목록
/highest-risk-area — 최고 위험 지역 상세
